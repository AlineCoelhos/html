    Clients

    é a parte dedicada ao público em que está sendo atendido, que consome a informação do servidor

    Clients são computadores numa rede cliente-servidor que são totalmente dependentes de um servidor para realizar suas atividades.

    Fatores relacionados ao Clients: HTML, CSS, JAVASCRIPT, imagens e arquivos estáticos.

    Linguagens de programação do Client: JAVASCRIPT

    Cache é uma área de memória disponível para que os clientes possuam parte dos arquivos da página em sua máquina.

    É a parte em que os servidores atribuem informações para seus clientes.

    Linguagens de programação do SERVERS: JAVASCRIPT, PHP, RUBY, C#, JAVA

    LAN É um tipo de conexão local em que a empresa prestadora do serviço para o cliente utiliza de forma direta sem uso da internet.

    Estrutura do HTML

    O importante é entender que HTML não é uma linguagem de programação, mas sim um corpo de textos que utiliza de atríbutos para formatação de sua página web.

    Um exemplo de software de uso do HTML: Visual Studio Code.

    Atributos ou tags são comandos que modificam seu texto, tendo cada um uma função.

    Lista ordenadas e não ordenadas

    Uma lista ordenada começa com a tag ol. Cada item da lista começa com a tag "li"

    Uma lista não ordenada é uma lista de itens. As listas de itens são marcadas com bullets (tipicamente pequenos círculos pretos). Uma lista não ordenada começa com a tag ul. Cada item da lista começa com a tag li.
